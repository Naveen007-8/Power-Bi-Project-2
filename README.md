# BMW Sales & Performance Dashboard 🚗

## Project Overview
This Power BI project provides a comprehensive analysis of BMW sales performance. It is designed to help stakeholders track key metrics, identify sales trends, and understand regional performance across different car models.

## Key Features
* **Sales Tracking:** Visualizes total revenue and units sold over time.
* **Model Analysis:** Compares performance between different BMW series and categories.
* **Regional Insights:** Drill-down capabilities to see which markets are driving growth.
* **Interactive Filters:** Users can filter by Date, Region, and Vehicle Category.

## Technical Skills Demonstrated
* **Data Modeling:** Created a star schema with Fact and Dimension tables (refer to the `.pbism` and `.tmdl` files).
* **DAX Measures:** Developed custom calculations for Year-over-Year (YoY) growth and Profit Margins.
* **Power Query:** Cleaned and transformed raw data for better reporting efficiency.
